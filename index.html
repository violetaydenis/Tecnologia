Tecnología

La tecnologia es el conjunto de conocimientos, herramientas y procesos que el ser humano crea para resolver problemas y mejorar su vida diaria, Desde el celular que usamos para comunicarnos hasta los avances en medicina, eduacion y transporte, todo tiene tecnologia detras. Sin la tecnologia, el mundo como lo conocemos no funcionaria igual. No se trata solo de tener los ultimos aparatos, si no de usarlos de forma responsable para aprender, crear y resolver problemas reales. Tambien se utiliza en la vida cotidiana para tener acceso a diferentes plataformas y lineas de internet ya que cada persona tiene un objetivo diferente para su propio uso cualquier usuario puede tener acceso a la tecnologia
